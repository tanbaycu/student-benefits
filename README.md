# Student Benefits & Lifetime Savings Kit ✦

Cổng thông tin tối giản tập hợp toàn bộ các ưu đãi sinh viên thực tế, chính thức trên toàn cầu và tại Việt Nam, kết hợp cùng bộ lập kế hoạch tiết kiệm trọn đời (Lifetime Savings Kit). Dự án được thiết kế theo phong cách tối giản Bauhaus/Swiss Grid phẳng, trực quan và mượt mà.

* **Tác giả:** **tanbaycu**
* **Trạng thái:** Hoạt động 24/7/365

---

## ✨ Tính Năng Nổi Bật

1. **127 Ưu Đãi Sinh Viên Thực Tế**:
   * Được chia thành 3 Theme Bento Grid:
     * **THEME 01 / TECH & SOFTWARE**: Các công cụ lập trình, thiết kế, AI, cơ sở dữ liệu.
     * **THEME 02 / ACADEMIA & GROWTH**: Các khóa học và giáo dục trực tuyến.
     * **THEME 03 / DAILY LIFE & TRAVEL**: Đời sống, ẩm thực, di chuyển, hàng không.
   * Phân trang tối giản 9 items/trang giúp giao diện sạch sẽ, dễ dùng.

2. **Hệ Thống Logo Sắc Nét & Vượt Qua AdBlocker**:
   * Tự động trích xuất tên miền và tải logo chất lượng cao từ **Google Favicon Service (sz=128)**.
   * Hoàn toàn không bị chặn bởi các phần mở rộng chặn quảng cáo (AdBlocker) như uBlock Origin, AdBlock Plus...

3. **Lifetime Savings Kit Dashboard (Apple Style Bottom Sheet)**:
   * Thêm các ưu đãi vào Kit cá nhân để tính toán lượng tài chính tích lũy trọn đời dự kiến.
   * Giao diện trượt mượt mà 60FPS từ đáy màn hình, bo tròn nghệ thuật.
   * Xuất bản in PDF chuyên nghiệp.

4. **Trải Nghiệm Tương Tác Cao Cấp (Emil Kowalski Inspired)**:
   * Thẻ sinh viên 3D tương tác (Magnetic/Tilt) phản hồi theo con trỏ chuột.
   * Cuộn trang siêu mượt bằng thư viện **Lenis**.
   * Loading Skeleton mượt mà khi chuyển danh mục.

## 🤝 Hướng Dẫn Đóng Góp (Contributing)

Dự án này được duy trì và phát triển nhờ sự đóng góp của cộng đồng sinh viên. Nếu bạn phát hiện ra bất kỳ ưu đãi sinh viên mới nào hoặc muốn cải tiến mã nguồn, hãy đóng góp theo các cách sau:

### 1. Đóng Góp Bằng Cách Mở Issue
Nếu bạn phát hiện ưu đãi sinh viên mới hoặc có ý tưởng tính năng:
1. Truy cập mục **Issues** trên kho lưu trữ GitHub của dự án.
2. Chọn **New Issue** (Tạo issue mới).
3. Cung cấp thông tin chi tiết:
   * Tên thương hiệu/ưu đãi.
   * Giá trị ưu đãi (Ví dụ: Giảm giá 50%, Miễn phí...).
   * Yêu cầu xác thực (Ví dụ: Email trường .edu, Thẻ sinh viên...).
   * Liên kết (URL) trang giáo dục chính thức của hãng.
4. Gửi issue. Đội ngũ cộng tác viên sẽ kiểm duyệt và cập nhật lên trang web trong thời gian sớm nhất.

### 2. Đóng Góp Trực Tiếp Bằng Pull Request (PR)
Nếu bạn muốn tham gia viết code:
1. **Fork** dự án này về tài khoản GitHub của bạn.
2. Tạo một nhánh (branch) mới: `git checkout -b feature/amazing-benefit`.
3. Thêm ưu đãi mới vào mảng dữ liệu `BENEFITS_DATA` trong file `src/App.jsx`.
4. Commit các thay đổi của bạn: `git commit -m 'Add new student benefit for XYZ'`.
5. Push nhánh lên GitHub: `git push origin feature/amazing-benefit`.
6. Mở một **Pull Request** giải thích chi tiết các thay đổi của bạn.

## 🚀 Hướng Dẫn Chạy Dưới Local

Đảm bảo bạn đã cài đặt NodeJS trên máy tính, sau đó thực hiện:

1. **Cài đặt các thư viện phụ thuộc:**
   ```bash
   npm install
   ```

2. **Khởi chạy môi trường phát triển (Dev Server):**
   ```bash
   npm run dev
   ```
   Trang web sẽ được chạy tại: `http://localhost:5173/`
